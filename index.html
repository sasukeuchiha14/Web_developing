CSS-Specificity

Important > Inline Style > ID (Last defined color will come) > Class/Attribute (Last defined color will come) > Element > Universal

  • Universal Selector: 0
  • Element selectors and pseudo-elements: 1
  • Class selectors, attribute selectors, and pseudo-classes: 10
  • ID selectors: 100
  • Inline styles: 1000

If we use 2 or more at a time, points will be added and color will be applied according to it.


For example: The specificity of

a. harryclass.rohan-class[href] :hover
